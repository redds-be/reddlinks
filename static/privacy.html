Privacy

This document concerns what data you provide to this website, the purpose of the data, how the data is stored, and how the data can be removed. No data is stored in your browser.

Data you directly provide

Information stored about a shortened link is limited to:

  • The URL you shorten which the shortened link will redirect you.
  • The custom path for the shortened link if any is set.
  • The length of the auto-generated shortened link.
  • The expiration time of the shortened link.
  • The password of the shortened link if any is set.

Data you passively provide

When you request any resource from this website (for example: a page, a font, an image, or an API endpoint) information about the request may be logged.

Information about a request is limited to:

  • The time the request was made.
  • The method of the request.
  • The requested URL. Note: This does not include the URL that the shortened link points to.

No identifying information is logged, such as the visitor's cookie, user-agent, or IP address. Here are a couple lines to serve as an example:

2023/12/13 14:35:01 GET /
2023/12/13 15:36:16 GET /nz_~Kx
2023/12/15 11:18:14 POST /

By default, reddlinks does not save any of these logs, whether these logs are saved depend on this site's administrator.
Please note that this log policy concerns the logs saved by the application itself and not the web server, the extent on which requests are logged depend on the web server's configuration used by this site's administrator

Removal of data

To remove the data you directly or indirectly provided, you can try to contact this site's administrator.
Please note that this site's administrator does not have the obligation to be contactable nor has the obligation to honor any requests.
